QuicSuite ドキュメント

インストール方法

macOS

  1. QuicSuite-1.0.0-installer.dmgをダウンロード
  2. DMGファイルをダブルクリックして開く
  3. Install_QuicSuite.commandをダブルクリック
  4. 管理者パスワードを入力
  5. インストール完了後、QuicPair.appが自動起動します

初回起動時の注意

macOSのセキュリティ機能により、初回起動時は以下の手順が必要です：

  1. QuicPair.appを右クリック
  2. 「開く」を選択
  3. 警告ダイアログで「開く」をクリック

基本的な使い方

iPhoneとのペアリング

  1. QuicPair.appで「Start Pairing」をクリック
  2. 表示されたQRコードをiPhoneで読み取り
  3. 自動的に安全な接続が確立されます

動作モードの変更

# 現在のステータスを確認
qvctl status

# プライベートモード（デフォルト）
qvctl mode set private

# フレンドモード（許可された端末のみ）
qvctl mode set friends

# パブリックモード（収益化可能）
qvctl mode set public

モデル管理

# 利用可能なモデルを確認
qvctl models list

# モデルをプリロード
qvctl models prefetch llama3.1-8b-instruct-q4k

アーキテクチャ

コンポーネント構成

  • QuicPair: macOS/iOSネイティブアプリ（Swift）
  • QUIVer-provider: P2Pネットワークデーモン（Go）
  • qvctl: コマンドラインツール

セキュリティ

  • エンドツーエンド暗号化（Noise Protocol）
  • macOS Keychainによる鍵管理
  • UDS + mTLSによるプロセス間通信
  • サンドボックス実行環境

トラブルシューティング

QuicPair.appが起動しない

システム環境設定 → セキュリティとプライバシー → 一般で「このまま開く」をクリック

ペアリングができない

  • 両デバイスが同じネットワークに接続されているか確認
  • ファイアウォール設定を確認
  • QuicPairを再起動してみる

qvctlコマンドが見つからない

echo 'export PATH="/usr/local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

アンインストール

sudo rm -rf /usr/local/quicsuite
sudo rm -f /usr/local/bin/qvctl
rm -rf /Applications/QuicPair.app
